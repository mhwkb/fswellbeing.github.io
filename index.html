How to help

If you are a user you can help out by

  • ..trying the software and suggest new features

  • ..trying the software and give feedback on technical problems

  • ..suggest new software

  • ..spread the word about completed software

If you are a programmer you can help by

  • ..improving the quality of the code

  • ..implement new features

If you are a designer or artist you can help by

  • ..suggesting how the design can be improved for existing software

  • ..increaing understanding of the user perspective
